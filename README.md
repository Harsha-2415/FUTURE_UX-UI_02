# 🚗 EcoRide - EV Rental App UI/UX

EcoRide is a modern Electric Vehicle Rental mobile application designed in Figma. The project focuses on providing a clean, user-friendly, and sustainable booking experience.

## 📱 Screens Included

- Splash Screen
- Onboarding Screen
- Login / Sign In
- Home Screen
- Vehicle Listing
- Booking Screen
- Booking Confirmation

## ✨ Features

- Clean and Modern UI
- Green Eco-Friendly Theme
- Easy EV Search & Booking
- Secure Payment Options
- Booking Summary
- Responsive Mobile Design
- User-Friendly Navigation

## 🎨 Tools Used

- Figma
- Material Icons
- Google Fonts
